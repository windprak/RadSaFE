# Software environment

The pipeline runs inside two Apptainer images defined in `environment/`.
This file lists, in order of decreasing relevance:

1. The Python packages the released source files actually import.
2. Phase 2 — exact pinned versions from `rag.def`.
3. Phase 3–5 — lock file (`inference_packages.lock.txt`) generated by
   `pip list` inside the built `inference.sif` image used for the paper.

## 1. Direct imports of the released code

Discovered by `grep -hE '^(import|from)' code_release/**/*.py` after
filtering the standard library:

| Package                                  | Used by                          | Purpose                                   |
| ---------------------------------------- | -------------------------------- | ----------------------------------------- |
| **numpy**                                | Phase 5                          | bootstrap arrays, percentiles             |
| **torch**                                | Phase 2                          | sentence-transformer backbone             |
| **transformers**                         | Phase 2, 4                       | tokenizers (BGE in Ph2, Mistral-Large in Ph4) |
| **sentence-transformers**                | Phase 2                          | BGE-large-en-v1.5 chunk embeddings        |
| **faiss-cpu**                            | Phase 2                          | flat inner-product index over Radiopaedia |
| **tiktoken**                             | Phase 2                          | GPT-4 tokenizer for chunk-size budgeting  |
| **openai** (client library)              | Phase 3, 4                       | vLLM speaks the OpenAI-compatible REST API; the `openai` Python client is used to talk to it. **Not** to OpenAI's hosted models. |
| **requests**                             | Phase 3                          | misc HTTP helpers                         |
| **tqdm**                                 | Phase 2, 3, 4                    | progress bars                             |

The standard library is otherwise sufficient (`os`, `sys`, `json`, `re`,
`math`, `csv`, `time`, `argparse`, `asyncio`, `pathlib`, `typing`,
`collections`, `itertools`, `functools`, `statistics`, `subprocess`,
`hashlib`, `warnings`, `random`).

## 3. Phases 3, 4, 5 (`environment/inference.def`)

Built on top of the public `vllm-openai_latest.sif` (vLLM's official
OpenAI-compatible image) with the additional pip installs declared in
`environment/inference.def`.

The complete frozen package list inside the built image is in
`inference_packages.lock.txt`, recorded with `pip list` inside the
built `inference.sif` (built on top of `vllm-openai_latest.sif`).

## Building the images

```bash
# Phase 2
apptainer build environment/rag.sif environment/rag.def

# Phases 3-5 (requires the upstream vLLM image at $MODELS_DIR)
apptainer build environment/inference.sif environment/inference.def
```

## Re-creating the lock file

```bash
apptainer exec environment/inference.sif pip list --format=freeze \
    > environment/inference_packages.lock.txt
```
